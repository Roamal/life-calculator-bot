# Life Calculator Bot 🕰️

Telegram бот который рассчитывает примерное количество дней 
оставшихся до конца жизни на основе статистики продолжительности 
жизни по стране и полу.

## Как использовать

1. Напиши `/start`
2. Введи дату рождения в формате `ДД/ММ/ГГГГ`
3. Выбери пол
4. Выбери страну

Бот рассчитает сколько дней тебе примерно осталось.

## Поддерживаемые страны

- Russia
- USA  
- Japan
- Germany
- World (среднемировая статистика)

## Стек

- Python
- python-telegram-bot
- Статистика продолжительности жизни ВОЗ

## Запуск

```bash
pip install python-telegram-bot
python main.py
```

## Команды

- `/start` — начать расчёт
- `/cancel` — отменить
- `/help` — помощь
